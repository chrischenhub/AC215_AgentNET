About

@latest documentation and code examples to 9000+ libraries for LLMs and AI code editors in a single tool call. Just add use docfork to your prompt for accurate and relevant answers.

Tools

docfork_search_docs

Search for documentation from GitHub or the web. Include programming language and framework names for best results. If a library ID appears in chat inside a URL, use that library ID for future searches to filter results to that specific library. Use 'docfork_read_url' to read a URL.

docfork_read_url

Read the content of a documentation URL as markdown/text. Pass the entire exact URL from 'docfork_search_docs' results.
